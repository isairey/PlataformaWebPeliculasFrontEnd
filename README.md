# 🌌 Acerca del proyecto

**React Movie App** es una aplicación web responsive desarrollada con React.js que consume una API de películas para mostrar información dinámica sobre películas populares, tendencias y detalles cinematográficos.

La plataforma permite:

- 🎬 Explorar películas
- 🔎 Buscar contenido
- ⭐ Ver ratings
- 🖼️ Mostrar posters dinámicos
- 📱 Diseño responsive
- ⚡ Navegación rápida
- 🌐 Consumo de APIs externas
- 🎞️ Visualización moderna

# 🚀 Configuración del proyecto

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/isairey/react-movie.git
```

---

## 2️⃣ Entrar al proyecto

```bash
cd react-movie
```

---

## 3️⃣ Instalar dependencias

```bash
npm install
```

---

## 4️⃣ Ejecutar proyecto

```bash
npm run dev
```

---

## 5️⃣ Abrir navegador

```bash
http://localhost:3000
```
